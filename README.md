# Retail Data Warehouse

Проект многослойного хранилища данных для анализа продаж и клиентов в ритейле.  
Реализован полный ETL-цикл: от загрузки сырых данных до построения аналитических витрин с использованием Airflow и dbt.

## Стек технологий

- **Docker** — контейнеризация всей инфраструктуры
- **Airflow** — оркестрация ETL-процессов
- **dbt (data build tool)** — трансформация данных, управление моделями, тестирование
- **PostgreSQL** — основная база данных
- **Python (pandas, psycopg2)** — кастомная загрузка данных

## Архитектура данных

Проект построен на многослойном подходе:

- **Raw** — сырые данные из CSV (загружаются в PostgreSQL)
- **Bronze** — исходные данные без изменений (источник для dbt)
- **Silver** — очищенные и структурированные данные (клиенты, заказы)
- **Gold** — агрегированные витрины для анализа (продажи по месяцам, метрики клиентов)

## Реализованные возможности

- Инкрементальная загрузка данных из CSV в PostgreSQL через Airflow
- Трансформация данных с помощью dbt (модели silver / gold)
- Тестирование качества данных:
  - встроенные тесты dbt (not_null, unique, relationships, accepted_values)
  - кастомные тесты на целостность и согласованность данных
- Документирование моделей через dbt docs
- Полная контейнеризация — проект поднимается одной командой

## Сильные стороны проекта

- **Контейнеризация** — весь стек развёртывается одной командой `docker-compose up -d`
- **Масштабируемость** — легко добавлять новые источники и метрики
- **Прозрачность данных** — полная трассировка от витрин до исходных систем
- **Data Quality** — автоматические тесты на всех этапах трансформации
- **Гибкость** — адаптация под конкретные бизнес-задачи

## Запуск проекта

```bash
git clone https://github.com/Deni1006/retail_dwh.git
cd retail_dwh
docker-compose up -d

После запуска доступны:

Airflow UI — http://localhost:8080 (admin / admin)

PostgreSQL — порт 5432 (airflow / airflow)

Запуск dbt-моделей (после старта контейнеров):
docker exec -it retail_dwh-dbt-1 dbt run
docker exec -it retail_dwh-dbt-1 dbt test
